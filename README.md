# LearnOps — Landing Page

AI-augmented study pipeline for the ADHD brain. Landing page with pre-order capture.

## Stack

- React 19 + Vite 8
- Tailwind CSS v4
- Lucide React (icons)
- Web3Forms (lead capture)
- Vercel (deployment)

## Development

```bash
npm install
npm run dev
```

## Deployment

Deploys automatically to Vercel on push to `main`.

Requires one environment variable in Vercel:

```
VITE_WEB3FORMS_ACCESS_KEY=your_key_here
```
